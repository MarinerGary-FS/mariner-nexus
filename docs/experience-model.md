# Mariner Nexus Experience Model

## Core Principle
The site should feel progressively more intelligent as the user interacts with it.

## Layer 1 — Trust (0–2 seconds)
- clean layout
- strong typography
- minimal motion
- clear messaging

## Layer 2 — Awareness (2–5 seconds)
- subtle glow activation
- soft movement
- depth cues
- responsive elements

## Layer 3 — Impact (5+ seconds)
- cinematic hero presence
- parallax interactions
- layered motion
- optional advanced rendering

## Goal
Create a moment where users think:
“What is this?”

Then immediately:
“This is a serious company.”

## Constraints
- never sacrifice clarity for effect
- never overwhelm the user
- motion must feel intentional, not decorative
