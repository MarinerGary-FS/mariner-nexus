# Credit Link System — Mariner Nexus

All client websites should include a subtle footer credit unless explicitly disabled by the client.

## Footer Credit
Text:
Built by Mariner Nexus — Digital Experience Systems

## Placement
- Bottom of footer
- Small and unobtrusive
- Must not distract from primary brand

## Behavior
- Opens in a new tab
- Slight opacity increase on hover
- Smooth transition

## Link Priority
1. Project-specific case study page on marinernexus.com
2. If no case study exists yet, link to marinernexus.com
3. If client requests removal, remove it without breaking footer spacing

## Visual Rules
- Text should remain elegant and subtle
- No badges, popups, or intrusive promotional elements
- Should feel like a signature, not an advertisement
